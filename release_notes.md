# Release Notes: Cora v1.0 (The Unified Workspace Update)

We are incredibly proud to announce the official release of **Cora v1.0**.

What began as a sleek Standard Calculator has evolved into a comprehensive, desktop-class computational environment designed for students, developers, and engineers. 

### What's New in v1.0
- **The Formula Library:** An interactive reference hub covering Mathematics, Physics, Finance, and Computer Science. Rendered beautifully with LaTeX, you can now seamlessly jump from a formula directly into the required computational workspace with a single click.
- **The Developer Workspace:** A massive new hardware-level calculator. Featuring a completely interactive 64-bit visual memory grid, live HEX/OCT/BIN/DEC/ASCII translation, and dynamic keyboard routing.
- **The Analytics Workspace:** Statistical Regression models and Time-Value of Money (TVM) models, beautifully visualized using fluid Recharts animations.
- **The Matrix Workspace:** Excel-style arrow-key navigation for ultra-fast $6 \times 6$ matrix entry.
- **The Graphing Engine:** Our custom 60FPS canvas engine allows for buttery smooth panning and zooming across multiple algebraic bounds simultaneously.

### Global Polish & Performance
- Complete responsive design overhaul.
- Upgraded `tsconfig.json` to ES2020 to support native JavaScript `BigInt` operations in the Developer Workspace, ensuring absolute precision for large memory addresses.
- Micro-interactions added to all sidebar elements, ensuring the app feels like a premium native OS application.

Thank you for being part of the Cora journey!
